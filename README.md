# Modern Font Stacks Mixins

[Iron Oxide Labs LLC](https://www.ironoxidelabs.com) presents sass mixins of [Modern Font Stacks](https://www.modernfontstacks.com) font stacks.

## Installation

`npm install modern-font-stacks-mixins`
